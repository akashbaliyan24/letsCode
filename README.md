## 🎯 Getting Started

### Prerequisites  
- Node.js (v16+)  
- Postrgres  
- Yarn or npm

### Setup

1. Clone the repository  
   ```bash
   git clone https://github.com/akashbaliyan24/letsCode.git
   cd letsCode
2. Backend setup 
  cd backend
  npm install
   create .env with required keys (e.g., PRISMA_URI, JWT_SECRET, PORT)
  npm run dev

3. Frontend setup
   cd frontend
  npm install
  npm run dev
then open http://localhost:5173 (or specified port)
